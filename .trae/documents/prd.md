# 优选商城前端产品需求文档

## 1. 产品概述
优选商城是一个基于Spring Cloud Alibaba微服务架构的电商平台前端，提供完整的用户购物体验，包括商品浏览、搜索、购物车、下单、支付、订单管理等功能。
- 目标用户：普通消费者和商城管理员
- 核心价值：提供美观、流畅的购物体验，支持完整电商业务链路

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户(USER) | 用户名注册 | 浏览商品、搜索、购物车、下单、支付、管理收货地址、查看订单 |
| 管理员(ADMIN) | 后台创建 | 商品管理、优惠券管理、订单发货 |

### 2.2 功能模块
1. **登录注册页**：用户登录、用户注册
2. **商城首页**：Banner轮播、分类导航、热门商品、推荐商品、新人优惠券
3. **商品列表页**：商品分类筛选、商品分页展示
4. **商品搜索页**：关键词搜索、分类筛选、价格区间筛选、排序
5. **商品详情页**：商品信息展示、加入购物车、立即购买
6. **优惠券中心**：优惠券列表、领取优惠券
7. **我的优惠券**：已领取优惠券列表、可用优惠券筛选
8. **购物车页**：购物车商品列表、数量修改、勾选/全选、删除、结算
9. **收货地址管理页**：地址列表、新增地址、修改地址、删除地址、设置默认地址
10. **订单确认页**：确认商品、选择地址、选择优惠券、价格计算、提交订单
11. **我的订单页**：订单列表、订单状态筛选、订单详情、支付、取消、确认收货
12. **后台管理页**：商品管理、优惠券管理、订单发货

## 3. 核心流程

### 3.1 购物主流程
用户浏览首页 → 点击商品进入详情 → 加入购物车/立即购买 → 进入购物车勾选商品 → 进入订单确认页 → 选择地址和优惠券 → 提交订单 → 模拟支付 → 管理员发货 → 用户确认收货 → 订单完成

```mermaid
    "选择地址" --> "提交订单"

```

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：#3B82F6（蓝色），体现科技感和信任感
- 辅助色：成功#22C55E、警告#F59E0B、危险#EF4444
- 背景色：#F6F8FB（浅灰蓝）
- 卡片背景：#FFFFFF
- 文字主色：#1F2937
- 文字次色：#6B7280
- 按钮风格：圆角(8px)、主色填充按钮、次要按钮描边
- 字体：系统默认无衬线字体，建立清晰的字号层级
- 布局：卡片式布局、柔和阴影、圆角(12px)、顶部导航栏
- 图标风格：线性图标，简洁明了

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 登录/注册 | 表单容器 | 白色卡片、居中布局、圆角、阴影 |
| 首页 | 导航栏 | 固定顶部、Logo、搜索框、购物车/用户入口 |
| 首页 | Banner | 圆角轮播图、指示器、左右箭头 |
| 商品卡片 | 商品项 | 圆角图片、商品名称两行截断、价格醒目色、销量灰色小字 |
| 购物车 | 表格/列表 | 复选框、商品缩略图、数量步进器、小计价格 |
| 订单卡片 | 订单项 | 状态栏、商品列表、价格区域、操作按钮组 |

### 4.3 响应式设计
- 采用桌面优先设计
- 主要适配1200px以上宽度桌面端
- 商品网格自适应列数
- 导航栏在小屏幕下折叠菜单

## 5. API接口说明
所有接口统一通过Gateway访问：http://localhost:9000
请求头需要携带：Authorization: Bearer {token}
